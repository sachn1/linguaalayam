AI Synthesis — optional

By default, search shows raw dictionary entries. Add your own API key to get a plain-language explanation written by an AI — like asking a question and getting a human-readable answer instead of a list of definitions.

Without a key: search still works fully. Quick, Exact, and Smart modes all return dictionary entries. Smart mode uses meaning-based search powered by a multilingual model running on our server — no key needed.

With a key: after search results are found, an AI reads them and writes a natural explanation in English. Your key is stored only in this browser and is sent directly to the AI provider with each request. It is never stored on our servers.
Get a key: Anthropic Console  ·  OpenAI Platform
Starts with sk-ant- for Anthropic or sk- for OpenAI. Stored only in your browser's localStorage.

About the corpora

lingua·ആലയം searches four open Malayalam corpora. You can filter by corpus in Search options on the main page.

Olam (English → Malayalam) — English words with Malayalam translations and definitions. From olam.in/p/open.

Datuk (Malayalam → Malayalam) — Malayalam headwords with Malayalam definitions. Useful for understanding a word in depth in Malayalam. From olam.in/p/open.

Shabdataaravali (Malayalam → Malayalam) — The authoritative Malayalam dictionary written by Sreekanteswaram Padmanabha Pillai in 1917, digitised by the Sayahna Foundation. Uses classical Malayalam orthography. Included automatically when you select Malayalam dictionary. From dict.sayahna.org.

Ekkurup (English thesaurus) — English synonym groups with Malayalam equivalents. Useful when you know the meaning but not the exact word. From olam.in/p/open.

For technical details and licence information see the user guide ↗.

Morphological analysis

Word forms shown alongside search results are analysed using mlmorph, a Malayalam morphological analyser developed by Swathanthra Malayalam Computing (SMC). It identifies the root word and grammatical features (tense, case, number) of inflected Malayalam words.
